# What is Alpha Vantage?
> A fast, lightweight Python interface to the AlphaVantage API

[Alpha Vantage](www.alphavantage.co) offers free realtime financial data and historical market data. This module implements a python interface to the free API provided by Alpha Vantage. This project requires a free API key, which can be created at their website(www.alphavantage.co).
